Задать температуру в градусах по Цельсию. Вывести в alert соответствующую температуру в градусах по Фаренгейту. Подсказка: расчет идет по формуле Tf = (9 / 5) * Tc + 32 , где Tf — температура по Фаренгейту, Tc — по Цельсию.

Код задания: 
    /**
     * Берет градусы по Цельсию в форме ввода на странице 
     * и выводит под полем ввода градусы по Фаренгейту
     * (округляет до 2 знака после запятой)
     */
    function celciusToFahrenheit() {
    let degCelcius = +document.getElementById("celcius").value;
    let text = "";
    if (isNaN(degCelcius) || (degCelcius < -273.15)) {
        text = "необходимо ввести число больше -273.15°C";
        }
    else {
        let degFahrenheit = (9 / 5) * degCelcius + 32;
        text = degFahrenheit.toFixed(2) + "°F";
        }
    document.getElementById("fahrenheit").innerHTML = text;
    }

Вывести в консоль значения выражений и объяснить почему получились такие значения, используя комментарии к каждому выражению

Код задания: 

    /**
     * Выводит в консоль результаты выражений в теле функции
     */
   function task_3() {
    console.log(10 + 10 + "10");//Ответ: строка "2010", (10+10)+"10", 
                                //неявное преобразование 20 к строке так как второй операнд строка

                                //так в каждой операции сложения один из операндов строка

                                //к числу из-за запятой, в результате любой операции с NaN получается NaN
